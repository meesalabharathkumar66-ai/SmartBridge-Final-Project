# Visualization Tool for Electric Vehicle Charge and Range Analysis

## 📌 Project Overview

The **Visualization Tool for Electric Vehicle Charge and Range Analysis** is a data analytics project developed using **Tableau**. The project analyzes electric vehicle (EV) data to provide meaningful insights into vehicle manufacturers, electric range, model year trends, vehicle types, and Clean Alternative Fuel Vehicle (CAFV) eligibility.

The dashboard enables users to interactively explore EV data using filters and visualizations, making it easier to identify trends and compare electric vehicles.

---

## 🎯 Objectives

- Analyze Electric Vehicle (EV) data.
- Create interactive visualizations using Tableau.
- Design a responsive dashboard.
- Identify trends in EV manufacturers, model years, and electric range.
- Provide insights through an interactive Tableau Story.

## 📈 Dashboard Insights

- Identifies the leading EV manufacturers.
- Shows the distribution of Battery Electric Vehicles (BEV) and Plug-in Hybrid Electric Vehicles (PHEV).
- Displays EV adoption trends over different model years.
- Analyzes electric driving range across vehicles.
- Visualizes county-wise EV registrations.
- Provides CAFV eligibility analysis.

## 🚀 How to Run the Project

1. Install Tableau Desktop or Tableau Public.
2. Open the `EV_Charge_Range_Analysis.twbx` workbook.
3. If required, reconnect the dataset.
4. Open the Dashboard tab.
5. Use the filters to interact with the visualizations.
